# Companion Flag Website — Design Brainstorm

## Context
A non-profit passion project celebrating human samenesses through a symbolic flag flown below all flags of the world. Founded 1999. Noble, humanistic, global, warm, inclusive.

## Idea A: "Parchment & Cartography" — Editorial Warmth

**Design Movement**: Arts & Crafts meets Editorial Print Design

**Core Principles**:
1. Aged parchment texture as the universal background — echoing the client's own slide aesthetic
2. Typographic hierarchy using a serif display font (Playfair Display or Cormorant Garamond) for headings, paired with a humanist sans (Source Serif 4) for body
3. Earth-toned palette: warm ivory, aged brown, muted gold, dusty slate blue
4. Map silhouettes as decorative motifs — echoing the geographic universality of the CF

**Color Philosophy**: Warm ivories and parchment tones evoke timelessness, history, and the handcrafted. Brown text (as the client uses in slides) feels intimate and personal, not corporate. Gold accents suggest value without ostentation.

**Layout Paradigm**: Asymmetric editorial columns — wide text blocks offset by full-bleed image panels. The "Such as?" slideshow occupies a full-width stage. Navigation is a slim top bar with serif wordmark.

**Signature Elements**:
1. Circular CF badge motif (matching the client's sticker design) used as section dividers
2. Aged paper texture overlay on all sections
3. Handwritten-style pull quotes in a calligraphic font

**Interaction Philosophy**: Gentle, deliberate. The "Such as?" button triggers a crossfade slideshow. Scroll-triggered fade-ins. No aggressive animations.

**Animation**: Slow crossfade between slides (800ms ease-in-out). Subtle parallax on hero. Section entrances with gentle upward drift.

**Typography System**: Cormorant Garamond (display, 48-72px) + Lora (body, 18px) + Playfair Display SC (section labels, uppercase, tracked)

## Idea B: "Humanist Modernism" — Clean Global Voice

**Design Movement**: Swiss International Style meets Humanist Web Design

**Core Principles**:
1. Clean white/off-white base with strong typographic hierarchy
2. Color drawn exclusively from the CF badge palette (sky blue, warm white, earth tones)
3. Photography-forward: full-bleed images of people, flags, and moments of connection
4. Grid-breaking asymmetry: text columns that don't align to a standard 12-column grid

**Color Philosophy**: The sky blue of the CF badge circles becomes the primary accent — referencing the actual flag design. Warm white backgrounds keep it accessible and non-corporate.

**Layout Paradigm**: Offset magazine layout — hero text bleeds left while imagery anchors right. Sections alternate between text-dominant and image-dominant. The "Such as?" section uses a horizontal scroll carousel.

**Signature Elements**:
1. Sky-blue horizontal rule as a section separator (referencing the CF stripe)
2. Large, bold numerals as section counters
3. Flag-inspired stripe motif: a thin colored stripe at the top of each page section

**Interaction Philosophy**: Confident and modern. Horizontal scroll for the samenesses gallery. Sticky navigation that changes color on scroll.

**Animation**: Slide-in from left for text, fade-up for images. Stripe color transitions between sections.

**Typography System**: DM Serif Display (headings) + DM Sans (body) — paired humanist fonts that feel both classic and contemporary

## Idea C: "Warm Globe" — Organic Worldly Elegance ← SELECTED

**Design Movement**: Organic Modernism meets World Heritage Editorial

**Core Principles**:
1. Warm, textured backgrounds inspired by the client's own slide aesthetic (parchment, world maps)
2. Deep, rich typography using Cormorant Garamond for display text — the same serif feel as the client's slides
3. The CF badge circle motif as a recurring visual anchor
4. Generous whitespace with asymmetric content placement — text left, imagery right (or full-bleed)

**Color Philosophy**: 
- Background: warm parchment (#F5EDD6 / oklch warm ivory)
- Primary text: deep brown (#3D2B1F)
- Accent: sky blue from CF badges (#87CEEB / light steel blue)
- Highlight: warm gold (#C9A84C)
- Section dividers: muted terracotta

**Layout Paradigm**: Narrative scroll — the page tells a story from top to bottom. Hero is full-width with the montage image. Each section flows naturally into the next with organic dividers (gentle wave or diagonal cuts). The "Such as?" slideshow is a full-width immersive panel.

**Signature Elements**:
1. World map watermark/ghost image behind key sections
2. CF badge circles as decorative elements and section markers
3. Warm parchment texture as the site-wide background

**Interaction Philosophy**: Warm and inviting. The "Such as?" button expands a full-width slide panel with prev/next navigation. Hover states use gentle color shifts. No jarring transitions.

**Animation**: Fade-up on scroll for text blocks. Crossfade for slideshow. Gentle parallax on the hero montage.

**Typography System**: 
- Display: Cormorant Garamond (matches client's slide aesthetic)
- Body: Lora (warm, readable serif)
- Labels/Nav: Raleway (clean, slightly geometric sans for contrast)

## Selected Design: **Idea C — "Warm Globe" Organic Worldly Elegance**

This design directly mirrors the aesthetic language already established in the client's slides — parchment backgrounds, world map silhouettes, Cormorant Garamond-style serif typography, and the sky-blue CF badge motif. It feels authentic to the project's spirit: warm, humanistic, global, and timeless.
